```md
### Webアプリの概要
「いつもの交通費メモ」は、保険営業マンなど外回りの仕事でよく訪問する顧客先や活動先の交通費を登録し、片道・往復の金額をすぐ確認できるアプリです。訪問先・出発地・到着地・片道料金を保存しておくことで、外出前や経費確認時に毎回乗換案内アプリで検索し直す手間を減らします。登録完了後には応援メッセージ画像を表示し、営業前に前向きな気持ちになれる体験も提供します。
```
